# Cash Control System — DGP Demo
## Sistema de Control de Efectivo para Personas Privadas de Libertad

Visual navigable demo for the General Directorate of Prisons of the Dominican Republic.

> Static frontend only — no backend, no real data, no form submissions. Ready for GitHub Pages.

## Modules

Dashboard · Administration (Prison Centers, Provinces, Account Types, Users & Roles) · Inmate Management (Registration, Account Opening, Balance Inquiry, Transaction History) · Transactions (Deposits, Withdrawals, Reversal) · Cash Desk Control (Opening, Closing, Reconciliation) · Special Processes (Transfer, Release, Family Delivery) · Security & Audit (Access Log, Audit, Roles & Permissions) · Reports

## Tech

Plain HTML5 + CSS3 + Vanilla JS. Zero dependencies. No build step.

## Multilanguage

ES (default) / EN — switcher in topbar, no page reload, preference saved in localStorage.

## Files

- `index.html` — app shell and all page sections
- `styles.css` — full styling with CSS variables and responsive layout
- `app.js` — translations, SPA navigation, language switcher, clock

## Deploy to GitHub Pages

Push to a repo → Settings → Pages → Deploy from branch `main` / root → Save.
